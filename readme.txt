Display a pre-publish reminder tasklist on the Posts Admin Panel

== Description ==

- display below Posts Admin textarea or sidebar
- paste html or php code snippets

**Demo:**
[Plugin Details & Demo](http://ericulous.com/2007/03/19/wp-plugin-genki-pre-publish-reminder/)

== Installation ==

1. Upload `genki_announcement.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure options in Admin Panel > Options > Pre.-Publish Reminder
